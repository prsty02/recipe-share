Project Recipe-share ini merupakan platform yang saya kembangkan untuk orang-orang dapat berbagi dan mendapatkan inspirasi berupa resep masakan yang dibagikan oleh sesama penggunanya. Antarmuka project ini dibangun dengan menggunakan Framework React.js dan framework css Tailwind.css. Untuk sisi back-end dari project ini didukung oleh noSQL database pada Firebase.

Untuk menjalankan project ini diperlukan beberapa persiapan, yaitu:
1. Ekstrak project
2. Instal NPM (jika NPM belum tersedia)
3. Jalankan command "npm install", untuk menginstall dependency yang dibutuhkan.
4. Jalankan command "npm start" project akan dijalankan pada post 3000
